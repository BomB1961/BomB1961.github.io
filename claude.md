# Rules
- Stack: Vanilla HTML/CSS/JS. No CDN library. No build.
- Deploy: GitHub Pages, main/docs.
- Data: repos.json, projects.json, 토큰은 design-tokens.css. 하드코딩 금지.
- API: 런타임 호출 금지. scripts/fetch-repos.js로 캐싱.

# Design (AI티 금지)
- 금지: 네온 퍼플/사이안, 그라디언트, 파티클, 타이핑, 글로우, 이모지, 영문감성카피, 3열동일카드, Orbitron.
- 톤: Unity 인디개발자. 한국어 본문. 차분한 에디토리얼.

# Quality
- 수정 전 파일 읽기. 추측 금지. 하드코딩 금지.
